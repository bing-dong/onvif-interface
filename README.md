# onvif-interface
onvif soap c++ interface

with a device discovery demo

## build

build with cmake

```
cd onvif
mkdir build
cd build
cmake ..
make
```

## run

```
cd build
./onvif
```

### demo output

`http://192.168.1.64/onvif/device_service http://[fe80::4ef5:dcff:fe9e:c86b]/onvif/device_service`
